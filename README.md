# Netherrack

## Dependencies
> ### [Tauri](https://tauri.app/)
> ### Rust
> ### Node.js

## How to build from source
> ### Linux (General)
>  - Clone the repository using `git clone https://github.com/BastionMC/Netherrack.git`
>  - Move to the directory using `cd Netherrack`
>  - Run the app with `npm run tauri dev`
